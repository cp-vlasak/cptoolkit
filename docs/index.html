How to Install

1

Download the ZIP

Click the download button above to get the latest version.

2

Extract the ZIP

Right-click the downloaded file and choose Extract All. Pick a permanent location — don't leave it in Downloads, because if you delete it the extension will break.

3

Open Chrome Extensions

Type chrome://extensions into your address bar and press Enter. You have to type it — clicking a link won't work.

4

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the page.

5

Load Unpacked

Click the Load unpacked button that appears in the top-left.

6

Select the Folder

Browse to the extracted folder (the one containing manifest.json) and select it. The extension should now appear in your toolbar.

Updating to a New Version

Download the new ZIP, extract it to the same folder (overwrite the old files), then go to chrome://extensions and click the refresh icon on the extension card.
